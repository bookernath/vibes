## Usage

Import the component.

<CodeBlock lang="bash">{`import { ProductsListSection } from '@/vibes/soul/sections/products-list-section'
`}</CodeBlock>

Query the appropriate data.

<CodeBlock lang="bash">{`const products = await getProducts()
const filters = await getFilters()
const sortOptions = await getSortOptions()
const breadcrumbs = await getBreadcrumbs()
const pagination = await getPagination()
`}</CodeBlock>

Display the component.

<CodeBlock lang="bash">
  {`<ProductsListSection
    title="Plants"
    breadcrumbs={breadcrumbs}
    products={products}
    totalCount={products.length}
    filters={filters}
    sortOptions={sortOptions}
    pagination={pagination}
/>`}
</CodeBlock>

### Streaming

The `products` property accepts either `Product[]` or `Promise<Product[]>`. By passing a `Promise<Product[]>`, you enable streaming which will show a loading state until the data is finished loading and streamed to the browser.

### Internationalization

Integrating internationalization into your Vibes components starts at the application level. You'll need a dictionary of strings that correspond to the properties of the `ProductsListSection` component. For example, the `title` property of the `ProductsListSection` component would have a corresponding key-value pair in the dictionary like so:

<CodeBlock lang="json">
  {`
{
  "ProductsListSection": {
    "title": "Hello world!",
    //...
  }
}`}
</CodeBlock>

To setup internationalization, we recommend using a package like [next-intl](https://next-intl-docs.vercel.app/). After it's configured, you can retrieve translated strings via the `useTranslations` hook. You can then pass the translated string into each component property appropriately.

<CodeBlock lang="tsx">
  {`import { useTranslations } from 'next-intl';
import { ProductsListSection } from '@/vibes/soul/sections/products-list-section;
 
export default function ProductListPage() {
  const t = useTranslations('ProductListSection');
  return (
    <div>
        <ProductListSection
            title={t('title')}
            // ...
        />
    </div>
  );
}`}
</CodeBlock>

For full setup instructions, refer to the [next-intl getting started docs.](https://next-intl-docs.vercel.app/docs/getting-started)
