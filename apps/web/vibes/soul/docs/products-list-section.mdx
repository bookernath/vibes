## Usage

Import the component.

<CodeBlock lang="bash">{`import { ProductsListSection } from '@/vibes/soul/sections/products-list-section'`}</CodeBlock>

Query the appropriate data.

<CodeBlock lang="bash">{`const products = await getProducts()
const filters = await getFilters()
const sortOptions = await getSortOptions()
const breadcrumbs = await getBreadcrumbs()
const pagination = await getPagination()
`}</CodeBlock>

Display the component.

<CodeBlock lang="bash">
  {`<ProductsListSection
    title="Plants"
    breadcrumbs={breadcrumbs}
    products={products}
    totalCount={products.length}
    filters={filters}
    sortOptions={sortOptions}
    pagination={pagination}
/>`}
</CodeBlock>

For full details, refer to the [API reference](#api-reference).

### Streaming

The `products` property accepts either `Product[]` or `Promise<Product[]>`. By passing a `Promise<Product[]>`, you enable streaming which will show a loading state until the data is finished loading and streamed to the browser.

## API Reference

### `ProductsListSection`

| Prop               | Type                                                                                                      | Default |
| ------------------ | --------------------------------------------------------------------------------------------------------- | ------- |
| `breadcrumbs?`     | [`Breadcrumb[]`](#breadcrumb)                                                                             |         |
| `title?`           | `Promise<string>` \| `string`                                                                             |         |
| `totalCount`       | `Promise<number>` \| `number`                                                                             |         |
| `products`         | [`Promise<Product[]>`](/docs/soul/product-card#product) \| [`Product[]`](/docs/soul/product-card#product) |         |
| `filters`          | `Promise<Filter[]>` \| `Filter[]`                                                                         |         |
| `sortOptions`      | [`Promise<SortOption[]>`](#sortoption) \| [`SortOption[]`](#sortoption)                                   |         |
| `compareProducts?` | [`Promise<Product[]>`](#product) \| `Product[]`                                                           |         |
| `pagination?`      | [`Promise<Pages>`](#pages) \| [`Pages`](#pages)                                                           |         |
| `compareLabel?`    | `string`                                                                                                  |         |
| `filterLabel?`     | `string`                                                                                                  |         |
| `sortLabel?`       | `string`                                                                                                  |         |
| `sortParam?`       | `string`                                                                                                  |         |

### `Breadcrumb`

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `label` | `string` |         |
| `href`  | `string` |         |

### `SortOption`

| Prop    | Type     | Default |
| ------- | -------- | ------- |

### `Pages`

| Prop            | Type     | Default |
| --------------- | -------- | ------- |

| `nextPage?`     | `string` |         |
