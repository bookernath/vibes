## Usage

<CodeBlock lang="bash">{`import { Cart } from '@/vibes/soul/sections/cart'`}</CodeBlock>

<CodeBlock lang="bash">{`const products = await getProducts()
const subtotal = await getSubtotal()`}</CodeBlock>

<CodeBlock lang="bash">{`<Cart
  title="Cart"
  lineItems={products}
  summary={{
    title: 'Summary',
    caption: 'Shipping & taxes calculated at checkout',
    subtotal: subtotal,
    subtotalLabel: 'Subtotal',
    shippingLabel: 'Shipping',
    taxLabel: 'Tax',
    tax: '$11.60',
    grandTotalLabel: 'Total',
    grandTotal: '$127.60',
    ctaLabel: 'Checkout',
  }}
  emptyState={{
    title: 'Your cart is empty',
    subtitle: 'Add some products to get started.',
    cta: {
      label: 'Continue shopping',
      href: '#',
    },
  }}
  removeLineItemAction={removeLineItemAction}
  updateLineItemQuantityAction={updateLineItemQuantityAction}
  // redirectToCheckoutAction={}
/>
`}</CodeBlock>

### Streaming

Notice in the [API Reference](#api-reference) that some properties accept data (ex. `string` or `number`) or a `Promise`. By passing a `Promise`, you enable the component to stream data from the server as it's loaded.

You can view this in the following example where the `lineItems` and `summary.subtotal` properties are passed as Promises:

## API Reference

### `Cart`

| Prop                           | Type                                                                                                    | Default  |
| ------------------------------ | ------------------------------------------------------------------------------------------------------- | -------- |
| `title?`                       | `string`                                                                                                | `"Cart"` |
| `lineItems`                    | [`CartLineItem[]`](#cartlineitem) \| [`Promise<CartLineItem[]`](#cartlineitem)>                         |          |
| `summary`                      | [`CartSummary`](#cartsummary)                                                                           |          |
| `emptyState`                   | [`CartEmptyState`](#cartemptystate)                                                                     |          |
| `removeLineItemAction`         | `function` <Tooltip content='(id: string) => Promise<void>' />                                          |          |
| `updateLineItemQuantityAction` | `function` <Tooltip content='({ id, quantity } : { id: string; quantity: number }) => Promise<void>' /> |          |
| `redirectToCheckoutAction`     | `function` <Tooltip content='() => Promise<void>' />                                                    |          |

### `CartLineItem`

| Prop       | Type                                                     | Default |
| ---------- | -------------------------------------------------------- | ------- |
| `id`       | `string`                                                 |         |
| `image`    | `Image` <Tooltip content="{src: string, alt: string}" /> |         |
| `title`    | `string`                                                 |         |
| `subtitle` | `string`                                                 |         |
| `quantity` | `number`                                                 |         |
| `price`    | `string`                                                 |         |

### `CartSummary`

| Prop               | Type                          | Default             |
| ------------------ | ----------------------------- | ------------------- |
| `title?`           | `string`                      | `"Summary"`         |
| `caption?`         | `string`                      | `"Receipt Summary"` |
| `subtotalLabel?`   | `string`                      | `"Subtotal"`        |
| `subtotal`         | `string` \| `Promise<string>` |                     |
| `shippingLabel?`   | `string`                      | `"Shipping"`        |
| `shipping?`        | `string`                      |                     |
| `taxLabel?`        | `string`                      |                     |
| `tax?`             | `string` \| `Promise<string>` |                     |
| `grandTotalLabel?` | `string`                      | `"Grand Total"`     |
| `grandTotal`       | `string` \| `Promise<string>` |                     |
| `ctaLabel?`        | `string`                      |                     |

### `CartEmptyState`

| Prop       | Type                                                        | Default |
| ---------- | ----------------------------------------------------------- | ------- |
| `title`    | `string`                                                    |         |
| `subtitle` | `string`                                                    |         |
| `cta`      | `object` <Tooltip content='{label: string, href: string}'/> |         |
